deployed link- https://client-psi-blond.vercel.app/

Follow the steps the see the code and run on local server-

1. clone the repository using the github url
2. change the directory-
```
cd client
```
3. Install the required npm packages
```
npm install
```
4. run the development server
```
npm run dev
```
5. open your browser and search for-
```
http://localhost:5173
```


Instructions-
1. This application is made for mobile screens only, use dev-tools to open the application in 375-425px
